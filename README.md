# Tax-Gap-Detection-and-Compliance-Validation-Service

Project Title: Tax Gap Detection & Compliance Validation Service

A Spring Boot backend system designed for tax auditors to validate financial transactions, compute expected taxes, detect compliance violations, execute configurable tax rules, generate exceptions,providing summary reports and maintain a complete audit trail.

Tech Stack:

1.Java 17

2.Spring Boot (3.x)

3.Spring Data JPA / Hibernate

4.MySQL

5.Maven

6.JUnit + Mockito
